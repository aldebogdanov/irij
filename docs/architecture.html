Evaluation Pipeline

Key insight: The lexer does the heavy lifting for Python-style indentation. IrijLexerBase post-processes every token from ANTLR's generated lexer, injecting synthetic INDENT/DEDENT tokens based on an indent stack. Parens suppress indentation tracking (newlines inside (), [], {} are ignored).

Environment & Scope Chain

Environment.java — lookup 
public Object lookup(String name, SourceLoc loc) {
    Cell cell = lookupCell(name);  // walk parent chain
    if (cell == null)
        throw new IrijRuntimeError("Undefined variable: " + name, loc);
    return switch (cell) {
        case ImmutableCell(var v) -> v;
        case MutableCell mc    -> mc.get();
        case VarCell vc        -> vc.get();
    };
}

private Cell lookupCell(String name) {
    Cell cell = bindings.get(name);
    if (cell != null) return cell;
    if (parent != null) return parent.lookupCell(name);  // ↑ recurse
    return null;
}

Cell Hierarchy — The Heart of Bindings

ImmutableCell

Local bindings & builtins. Cannot be reassigned or redefined.

record ImmutableCell(Object value)
    implements Cell {}
✅ x := 42 inside a function body
✅ println, head, true
❌ x <- 0 → error
MutableCell

Explicit mutable bindings via :!. Reassignable via <-.

class MutableCell implements Cell {
    private Object value;
    void set(Object v) { value = v; }
    Object get() { return value; }
}
✅ counter :! 0
✅ counter <- counter + 1
🔗 Closures capture by reference
VarCell Phase 2.5

Top-level rebindable. Clojure-style Var. Updated in-place on redef.

class VarCell implements Cell {
    volatile Object value; // thread-safe
    void set(Object v) { value = v; }
    Object get() { return value; }
}
✅ fn f (x -> x + 1) at top level
✅ fn f (x -> x * 10) redefinition
❌ f <- ... → error (use :=)

Hot Redefinition — Why VarCell Matters

Interpreter.java — defineInScope 
/** VarCell at global scope, ImmutableCell locally. */
private void defineInScope(Environment env, String name, Object value) {
    if (env == globalEnv) {
        env.defineVar(name, value);   // updates in-place if exists
    } else {
        env.define(name, value);      // local: ImmutableCell
    }
}

Function Application

// Interpreter.apply()
case Lambda lam -> {
    if (args.size() < lam.arity())
        yield new PartialApp(lam, args); // curry!
    var callEnv = lam.closure().child();
    // bind params via pattern matching
    for (int i = 0; i < lam.params().size(); i++)
        matchPattern(lam.params().get(i), args.get(i), callEnv);
    yield eval(lam.body(), callEnv);
}
